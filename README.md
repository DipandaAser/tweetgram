# tweetwatcher

A simple Telegram bot that send you a screenshot when there is a new tweet with some specified hashtag.


All think you have to do is to edit the _config.json_ file and place it in the same folder.

**Example :**
```json
{
    "Hashtag": "#caparledev",
    "BotToken": "1976ERBNRIJNFIREUHFERFERFP",
    "Port": "8080",
    "PublicURL": "bot.telegram.caparledev.io"
}
```
